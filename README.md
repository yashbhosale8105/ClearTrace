## 🚀 Core Features

### 1. 🔍 Intelligent UPI Investigation
- **Real-Time Anomaly Detection**: Automatically flags transactions based on Device ID mismatches, IP address anomalies, Location shifts, and High-Amount thresholds.
- **Rapid-Fire Detection**: Identifies "velocity attacks" where multiple transactions occur in a suspiciously short window.

### 2. 🤖 Gen-AI Investigation Engine (Gemini 2.0)
- **AI-Powered Reasoning**: Uses Google Gemini to analyze raw transaction data and generate human-readable, professional investigation reports.
- **Outcome Classification**: Determines if a case is `SUSPICIOUS`, `LEGITIMATE`, or `REQUIRES_FURTHER_REVIEW` with detailed justification.

### 3. 🏦 Cheque Analysis & OCR
- **Vision AI**: Analyzes cheque images to extract Account Numbers, IFSC codes, Amounts, and Signatory details.
- **Validation**: Cross-references OCR data with bank records to detect forged signatures or insufficient balances.

### 4. 📊 Multi-Dimensional Dashboards
- **Vibrant UI**: A "Cyber-Noir" inspired dashboard built with React and Vite.
- **Data Viz**: Real-time distribution charts for risk levels, anomaly types, and merchant volume.

### 5. 🔊 Voice & Reports
- **Voice Summary**: Listen to investigation results via AI-generated voice output (gTTS).
- **PDF Generation**: One-click generation of formal, bank-ready investigation reports.

### 6. 💬 Interactive Chatbot
- A central nexus for officers to query the database, investigate specific UPI IDs, and interact with the AI assistant.

## 🛠️ Tech Stack

### **Dimension A: The Backend**
- **Language**: Python (Flask)
- **Database**: SQLite
- **Processing**: Pandas (for data manipulation)
- **PDF Engine**: FPDF2
- **Voice Engine**: gTTS

### **Dimension B: The Frontend**
- **Modern Hub**: React.js with Vite
- **Styling**: Vanilla CSS (Custom Cyber-Noir Theme)
- **Icons**: Lucide React
- **Charts**: Chart.js

### **Dimension C: The Intelligence**
- **Model**: Google Gemini Pro & Flash
- **Vision**: Gemini Multi-modal for Cheque OCR

## 🔌 Setup & Installation

### **1. Clone the Multiverse**
```bash
git clone https://github.com/yashbhosale8105/ClearTrace.git
cd ClearTrace
```

### **2. Setup Backend (Dimension A)**
```bash
# Create Virtual Environment
python -m venv venv
source venv/bin/activate  # Windows: .\venv\Scripts\activate

# Install Dependencies
pip install -r requirements.txt

# Run Backend
python app.py
```

### **3. Setup Frontend (Dimension B)**
```bash
cd frontend
npm install
npm run dev
```
